PyWPS-4 REST interface

This is a HOWTO for the REST interface of PyWPS.

GET PUT POST DELETE
{{ rest_url_config }} Returns a list of all server instance configuration - - -
{{ rest_url_config_id }} Returns the configuration of serverid server instance Changes specific configuration of serverid server instance.
(Passed data must be JSON formatted and submitted with Header Content-Type application/json)		 - -
{{ rest_url_server }} Returns a list of server instances containing information about the host, port, pid and process status - - -
{{ rest_url_server_id }} Returns the host, port, pid and process status of serverid server instance. Creates and starts a new serverid server instances specified by the passed data.
(Passed data must be JSON formatted and submitted with Header Content-Type application/json)		 - Stops and removes serverid server instances from currently available server instances.
{{ rest_url_server_processes }} Get list of all the processes activated on the specified server - - Deactivate/Remove all the processes from the specified server
{{ rest_url_server_process_id }} Activate the specified process for the specified server - - Deactivate/Remove the specified server process
{{ rest_url_server_process_name }} Activate the specified process for the specified server - - Deactivate/Remove the specified server process
{{ rest_url_server_pause }} Pauses the server instance specified by serverid. - - -
{{ rest_url_server_resume }} Resumes the server instance specified by serverid. - - -
{{ rest_url_pywps_processes }} Returns a JSON formated list with all the PyWPS processes available. - - Completely removes all the PyWPS processes available. (Use with Caution!! The files will be gone for good!)
{{ rest_url_pywps_process }} Returns the source code of the specified PyWPS process. Updates the specified PyWPS process. This will fail if the process does not exist. Creates or updates the specified PyWPS process. Deletes the specified PyWPS process.

PyWPS documentation is at pywps.rtfd.org; source code and issue tracking at github.com/jachym/pywps-4. Code for this app is at github.com/jachym/pywps-4/tree/master/demo.
